# Prisma Setup Prompt

Usando el contexto del proyecto, configurar Prisma.

Requisitos:
- Definir modelos según database.md
- Incluir:
  - Product
  - User

Generar:
- schema.prisma
- prisma.service.ts

Reglas:
- No agregar modelos extra
- No relaciones complejas aún
- Código listo para ser usado por servicios

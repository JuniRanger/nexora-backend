# Extend Prompt

Extiende el módulo existente siguiendo el contexto de Nexora ERP.

Requisitos:

- Mantener Clean Architecture
- No romper la estructura actual
- Reutilizar interfaces existentes
- Mantener inversión de dependencias

Acciones:

- Identificar dónde encaja la nueva lógica
- Crear nuevos use-cases si es necesario
- Extender interfaces si aplica
- Implementar en infrastructure

Explicar:

- Qué se agregó
- En qué capa se agregó
- Por qué se hizo así

Restricciones:

- No modificar lógica existente innecesariamente
- No sobre-ingeniería
